**Project Overview**
This dashboard analyzes sales data from 2020–2022 using 3 structured Excel tables.
The report dynamically updates based on slicer selections, allowing users to explore trends, performance, and key metrics interactively.

**Tools & Technologies Used**

1. MS Excel – Data Cleaning & Preprocessing
2. Power BI – Data Modeling & Visualization
3. DAX – KPI & Measure Creation

**Dataset Information**

1. 3 Dummy Sales Data Tables (Excel)
2. Cleaned and formatted before importing into Power BI
3. Data includes:
    Sales
    Cost
    Profit
    Quantity
    Transactions
    Category
    Brand
    State

**Key Features**

✔ Interactive & Responsive Dashboard
✔ Dynamic KPI Cards (Sales, Profit, Quantity, Transactions)
✔ Category-wise Performance Analysis
✔ Brand-wise Contribution
✔ State-level Sales Visualization
✔ Slicers for filtering data dynamically

**Business Insights Generated**

1. Identified top-performing product categories
2. Analyzed brand contribution to total revenue
3. Compared yearly sales trends
4. Evaluated regional sales performance
